# sensor-read-data-temperature-humidity
Дополнения к итоговой версии:
- Реализовано хранение данных получаемых от пользователя по http-запросу(имя, пароль сети) во Flash памяти контроллера(NVS)
- Реализовано получение/чтение данных из Flash памяти
- Добавлен механизм прерываний при котором каждые 30 минут идет проверка на подключение к сети
- Использован еще один таймер, который отвечает за прерывания и проверку на наличие wifi
