Проект на esp32 для чтения данных с датчика температуры и влажности.
Реализовано: 
- сервер
- точка доступа
- протокол modbus
- считывание по прерываниям таймера
- обработка нажатий на кнопки по прерыванию
- отрисовка дисплея
- инициализирована файловая система
- запись данных в файловую систему
- передача данных из файловой системы клиентам по http-протоколу
- получение данных о времени через ntp-сервер
- обработка подключения к сети wifi
- получение времени с сервера при помощи js
- получение данных о сети от клиента (имя сети и пароль сети)
- отправка данных на гугл таблицы
